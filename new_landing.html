Selected Writing

Geometric Search Trees

A family of randomized set data structures that generalizes the zip-trees to work more efficiently in the presence of cache hierarchies. Can be used for set fingerprinting and for membership proofs.

Range-Based Set Reconciliation

Range-based set reconciliation is a simple approach to efficiently computing the union of two sets over a network, based on recursively partitioning the sets and comparing fingerprints of the partitions to probabilistically detect whether a partition requires further work. Quite a few open source projects started considering this algorithm.

Strict Principles for Lazy Sequences

APIs like iterators, streams, sinks, readers, writers, are all APIs for lazily working with potentially infinite sequences of items. There is a large number of competing designs, but little principled work on evaluating or deriving these designs. Starting from the question of how to objectively evaluate such APIs, I derive some instructive and quite useful hierarchies of interfaces for working with sequences.
